# 一格 Uno

`一格 Uno` 是一个面向 Windows 的本地桌面文件分类整理工具。它会扫描你选择的源目录，自动提取文件名里的高频字段，帮助你生成规则；在真正移动文件之前先生成预览计划，确认无误后再执行整理。

当前版本基于 `Electron + React + TypeScript + Tailwind`，界面延续了浅色、克制、分区明确的 GitHub 风格，但强化了工作台感和品牌视觉。

## 当前能力

- 支持选择源目录和输出根目录
- 支持源目录与输出根目录相同；相同时会自动跳过已整理子目录
- 自动统计高频字段，并支持一键生成规则
- 规则支持：
  - 包含关键词
  - 排除关键词
  - 扩展名限制
  - 任一关键词命中 / 全部关键词命中
- 支持拖动规则改变优先级
- 支持规则搜索、筛选、分组、批量启用、批量停用、批量删除
- 支持规则导入 / 导出
- 执行前先生成预览，不会直接动文件
- 目标目录存在重名文件时自动改名，例如 `文件 (1).pdf`
- 跨盘移动失败时自动回退为“复制后删除”
- 整理过程中显示进度、当前文件，并支持取消
- 支持撤销最近一次成功整理
- 保存本地规则、最近目录和最近任务历史

## 当前限制

- 仅支持 Windows
- 仅支持本地磁盘目录
- 不支持后台监听
- 不支持网盘、SMB、NAS 等特殊路径
- 默认处理所有文件类型，不做全局文件类型过滤

## 使用流程

1. 启动 `一格 Uno`
2. 在首页选择源目录
3. 选择输出根目录
4. 查看自动识别出的高频字段，并勾选生成规则
5. 在规则管理页补充或调整规则优先级
6. 回到首页生成预览计划
7. 确认目标路径、自动改名结果和未命中文件列表
8. 点击“确认后开始整理”
9. 在首页查看最近任务，必要时执行撤销

## 规则示例

| 规则名称 | 包含关键词 | 排除关键词 | 扩展名 | 输出子文件夹 |
| --- | --- | --- | --- | --- |
| 合同 | 合同, agreement, contract | 草稿, draft | pdf, docx | 合同 |
| 发票 | 发票, invoice | 作废 | pdf | 发票 |
| 素材图 | product, banner | psd-temp | jpg, png, webp | 素材图 |

## 本地数据

程序会在 Electron 的 `userData` 目录保存：

- 规则列表
- 最近使用的源目录和输出目录
- 最近任务历史

通常位于：

```text
%APPDATA%/Uno
```

## 目录结构

```text
src/
  main/       Electron 主进程、IPC、任务执行、历史存储
  preload/    安全桥接层
  renderer/   React 界面、页面与组件
  shared/     共享类型、规则工具、匹配与建议逻辑
```
